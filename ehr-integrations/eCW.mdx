### Key Metrics:

- 3 Days Go-Live date upon Launch checklist completion
- 10 Seconds Average Time for a Note to copy into EHR/Chart (Sync)
- 99% Sync Reliability

### Launch Checklist:

- Three logins with admin + provider permissions
- Turn off CAPTCHA + Cloudflare
- Standard 2FA setup with a one-time code
- All permissions granted (read/create/edit)
- Identify at least one Test Appointment
- Identify a Test Patient
