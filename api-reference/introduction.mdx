## Overview

The Ambient AI API v2 enables you to integrate ambient AI documentation capabilities into your applications. With this API, you can:

- **Authenticate** using OAuth2 password flow
- **Transcribe** audio files into text
- **Manage templates** with JSON Schema for clinical note structures
- **Generate clinical notes** from transcriptions using customizable templates

## Base URL

All API requests should be made to:

```
https://api.sandbox.commure.com/ambient/scribe
```

## API Versioning

The API uses versioning in the URL path. This is version `v1`, which uses the following path pattern:

```
/v1/{endpoint}
```

## Authentication

The API uses JWT (JSON Web Token) authentication. To authenticate:

1. Obtain a JWT token using the authentication endpoint
2. Include the token in the `Authorization` header for all subsequent requests:

```
Authorization: Bearer {your_jwt_token}
```

## Request Format

- Authentication endpoint accepts form data (`application/x-www-form-urlencoded`)
- Transcription endpoint accepts multipart form data (`multipart/form-data` for file uploads)
- Template endpoints accept JSON (`application/json`)

## Response Format

All responses are returned in JSON format. Success responses include relevant data, while error responses include error details in a consistent format.

## Rate Limits

API rate limits may apply depending on your subscription tier.
