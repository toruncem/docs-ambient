## Supported Platforms

- Windows 10 or Windows 11
- Require Internet connection (specifically for low-latency, medically accurate transcription)
- Microphone (built-in or external)

## EHR Compatibility

Commure Dictation works with **any EHR or application** that accepts text input. No special integration or IT setup is required.

**Tested EHRs include:**
- Epic
- Cerner
- Athenahealth
- NextGen
- eClinicalWorks
- Practice Fusion
- WebPT
- AdvancedMD

If your EHR has a text field, Commure Dictation works with it. If you face any issues, reach out to our [team](support@ambient.commure.com), and we will ensure it is resolved.

## Citrix Virtual Desktop Support

For organizations using Citrix Workspace to access their EHR, Commure Dictation includes dedicated virtual channel integration. This ensures reliable text injection into applications running within your Citrix session.

## Not Currently Supported

- Offline mode (internet connection required)
- macOS and Linux
