---
title: "Voice Commands"
description: "Complete reference for punctuation, formatting, and special character voice commands in Commure Dictation."
---

Commure Dictation recognizes voice commands for punctuation, formatting, and special characters. Simply speak the command as part of your dictation.

## Punctuation

| Say This                                  | Inserts |
| ----------------------------------------- | ------- |
| "period", "full stop", or "dot"           | `.`     |
| "comma"                                   | `,`     |
| "question mark"                           | `?`     |
| "exclamation mark" or "exclamation point" | `!`     |
| "colon"                                   | `:`     |
| "semicolon"                               | `;`     |
| "ellipsis"                                | `...`   |

## Quotes and Apostrophes

| Say This                      | Inserts |
| ----------------------------- | ------- |
| "open quote" or "begin quote" | `"`     |
| "close quote" or "end quote"  | `"`     |
| "apostrophe"                  | `'`     |
| "apostrophe ess"              | `'s`    |

## Brackets and Parentheses

| Say This                             | Inserts |
| ------------------------------------ | ------- |
| "open parenthesis" or "open paren"   | `(`     |
| "close parenthesis" or "close paren" | `)`     |
| "open bracket"                       | `[`     |
| "close bracket"                      | `]`     |
| "open brace"                         | `{`     |
| "close brace"                        | `}`     |

## Numbered Lists

Create auto-formatted lists. 

| Say This                                                                      | Inserts                                         |
| :---------------------------------------------------------------------------- | :---------------------------------------------- |
| "Create numbered list"<br />"Start numbered list"<br />"Insert numbered list" | Starts a list, beginning with "1."              |
| "Next item"<br />"Next number"<br />"Number 3"                                | Adds item to the numbered list after a new line |
| "End list"                                                                    | Ends the numbered lists                         |

## Fields and Lines

| Say This                  | Inserts                      |
| ------------------------- | ---------------------------- |
| "new line" or "next line" | Line break                   |
| "new paragraph"           | Double line break            |
| "Next field"              | Goes to next placeholder     |
| "Previous field"          | Goes to previous placeholder |

## Numbers and Currency

| Say This       | Inserts |
| -------------- | ------- |
| "50 dollars"   | `$50`   |
| "100 euros"    | `€100`  |
| "percent sign" | `%`     |
| "degree sign"  | `°`     |
